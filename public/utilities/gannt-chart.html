Hướng dẫn sử dụng (có subtasks)

  • Cấu trúc parent-child: Dùng cột "Parent ID" để chỉ định task cha
  • Task cha: Parent ID = 0 hoặc để trống
  • Task con: Parent ID = ID của task cha
  • Thay đổi cột: Kéo đường viền bên phải tiêu đề cột để resize
  • Di chuyển task: Kéo task để thay đổi timeline
  • Fit view: Nhấn nút "Fit View" để hiển thị toàn bộ timeline
